# РегионМонитор — verified sources

Статический дашборд для GitHub Pages с только подтверждёнными источниками.

## Что включено
- Москва: подтверждённый Telegram-канал Москва 24
- Санкт-Петербург: подтверждённый RSS Фонтанки
- Ленинградская область: 47news как HTML fallback и подтверждённый Telegram-канал

## Запуск
Откройте `index.html` локально или загрузите проект в GitHub и включите GitHub Pages.

## Примечание
В этом архиве оставлены только источники, которые были отдельно подтверждены при проверке ссылок. Telegram-каналы показаны как внешние ссылки без автоматической RSS-ленты.
